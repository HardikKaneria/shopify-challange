# Best Sellers Product Carousel

Responsive product carousel built with HTML, TailwindCSS, and Vanilla JavaScript.

## Features

- 10 product cards
- Responsive desktop horizontal carousel
- Mobile 2-column grid
- Mobile-only Show More button
- Smooth reveal animation for hidden mobile products
- Image hover swap using secondary product images
- Custom draggable scrollbar
- Scrollbar thumb grows on hover/touch
- Semantic HTML structure
- No framework or build step required

## Tech Stack

- HTML
- TailwindCSS
- Vanilla JavaScript

## Preview

Open `index.html` directly in the browser.
